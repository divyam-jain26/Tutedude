# Task 1
**Functionality of the Program**

This program **calculates the factorial of a given number**. Its functionality is:

1. It **defines a function `factorial(num)`** that computes the factorial of a number.

   * The variable `result` is initialized to 1.
   * A `for` loop multiplies `result` by every integer from **2 to `num`**.
   * The final factorial value is returned.
2. The program **takes an integer input** from the user.
3. It **calls the `factorial` function** with the user-provided number.
4. It **displays the factorial result** using formatted output.

In summary, the program accepts a number from the user and prints its **factorial**, calculated using an iterative approach.
