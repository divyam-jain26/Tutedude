# Task 1
This program acts as a basic calculator. Its functionality is as follows:

1. It takes two numerical inputs from the user (a and b) and converts them into floating-point numbers.

2. It performs four arithmetic operations on the given numbers:

- Addition (a + b)
- Subtraction (a - b)
- Multiplication (a * b)
- Division (a / b)

3. It displays the result of each operation with a descriptive label using formatted output.

In summary, the program reads two numbers from the user and prints their sum, difference, product, and quotient.
