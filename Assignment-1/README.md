# Task 1
**Functionality of the Program**

This program acts as a **basic calculator**. Its functionality is as follows:

1. It **takes two numerical inputs** from the user (`a` and `b`) and converts them into floating-point numbers.
2. It **performs four arithmetic operations** on the given numbers:

   * **Addition** (`a + b`)
   * **Subtraction** (`a - b`)
   * **Multiplication** (`a * b`)
   * **Division** (`a / b`)
3. It **displays the result of each operation** with a descriptive label using formatted output.

In summary, the program reads two numbers from the user and prints their **sum, difference, product, and quotient**.

# Task 2
**Functionality of the Program**

This program performs **user input handling and string concatenation**. Its functionality is:

1. It **accepts the user's first name** and stores it in the variable `first`.
2. It **accepts the user's last name** and stores it in the variable `last`.
3. It **concatenates the first name and last name** with a space in between to form the full name.
4. It **displays a personalized greeting message** using formatted string output.

In summary, the program takes the user's name as input and prints a **welcome message addressing the user by their full name**.
